Attributes

Information regarding negative events such as deep discharges, overcharges, and accidents.

governedBy: Annex XIII (4d); Annex VII Part B (4)

Properties

Property Description Type governedBy
Number of deep discharge events (dppk:eventsDeepDischarge) The number of occasions in which voltage has dropped below the lower operational limit. xsd:integer Annex VII Part B (4)
Number of overcharge events (dppk:eventsOvercharge) The number of occasions in which voltage has increased above the upper operational limit. xsd:integer
Information on accidents (dppk:eventsAccidents) Information and data resulting from use such as accidents, based on measuring abrupt positive or negative acceleration. xsd:string Annex XIII (4d)
