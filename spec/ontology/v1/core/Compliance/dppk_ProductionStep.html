A significant step in the manufacturing or processing chain of the product, including its location.

Properties

Property Description Type equivalentProperty
Production Step Type (dppk:productionStepType) The name or description of the production step (e.g., Assembly, Weaving, Smelting).
Production Location Country (dppk:productionLocationCountry) The country where the production step took place. Recommended to use ISO 3166-1 alpha-3 codes. gs1:countryCode
